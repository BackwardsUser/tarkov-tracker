# Tarkov Quest Tracker
This website is used to track tarkov quests for users,
it allows users to track their quests easier than other websites.
You can track specifically kappa required quests, and you can see the quests in a flowchart view.

This application was vibe coded, expect issues.

Note: I don't like to make vibe coding a habit, but needed something functional quick, thought it was pretty cool and helpful, so decided to publish it. :)
